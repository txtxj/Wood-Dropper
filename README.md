关卡文件格式：

若干个 byte 数字，

前两个数字分别表示列数与行数，对于后续数字：

 - 数字 `x` 为正表示该位置有一个长度为 `x` 的木块；
 - 数字 `x` 为负表示该位置有一个长度为 `-x` 的空缺；
 - 数字 `x` 为 0 表示换到下一行（上一层）。
